# HardLinkCopier (Windows Utility)

Программа для копирования файлов с сохранением структуры каталогов и создания жёстких ссылок на копии файлов в другом каталоге. Подходит для NTFS-дисков в Windows.

## 📌 Возможности

- Рекурсивный обход исходной директории
- Копирование файлов в целевой каталог с сохранением структуры
- Создание жёстких ссылок (hard links) на копии
- Поддержка десятков тысяч файлов
- Оценка производительности (время и скорость создания ссылок)

## 🛠️ Сборка

**Требования:**
- Windows
- C++17
- Компилятор (MSVC, MinGW)

**Компиляция через g++:**
```bash
g++ src/hardlink_demo.cpp -o demo.exe
g++ src/scan_directory.cpp -std=c++17 -o scan.exe
g++ src/copy_files.cpp -std=c++17 -o copy.exe
g++ src/create_links.cpp -std=c++17 -o link.exe
g++ src/measure_links.cpp -std=c++17 -o measure.exe
```

## 🔍 Примеры использования

### 1. Копирование файлов

```cpp
// sourceDir: C:\Input
// copyDir:   C:\Copied

copy_all_files("C:\Input", "C:\Copied");
```

### 2. Создание жёстких ссылок

```cpp
// copyDir:   C:\Copied
// linkDir:   C:\Links

create_hard_links("C:\Copied", "C:\Links");
```

### 3. Оценка скорости

```cpp
create_links_with_timer("C:\Copied", "C:\Links");
```

## ⚠️ Ограничения

- Работает только на NTFS
- Требуются права доступа к файлам
- Жёсткие ссылки можно создавать только в пределах одного тома
